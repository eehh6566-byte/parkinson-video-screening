# 帕金森震颤视频筛查静态演示网站

这是一个可部署到 GitHub Pages 的纯静态页面。

## 页面内容

- 患者视频输入与本地预览
- 双手视频筛查流程演示
- 右手/左手震颤频率展示
- 帕金森震颤风险提示
- 视频分析与 Xsens 金标准的研究验证对比表

## 重要说明

真实患者使用时只需要上传视频，不需要上传 Xsens 金标准文件。

金标准对比表只用于算法验证展示，说明视频算法与传感器结果的接近程度。

当前版本是静态演示页，点击“开始筛查分析”后展示预设样本结果；如果要接入真实算法，需要把按钮逻辑改为调用后端 API。

## GitHub Pages 部署

1. 把本目录内容上传到 GitHub 仓库。
2. 在仓库 Settings -> Pages 中启用 Pages。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，目录选择 `/root`。
5. 保存后访问 GitHub Pages 地址。
